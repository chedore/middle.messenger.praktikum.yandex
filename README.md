## Описание

Проект веб мессенджера.

## Установка и запуск

- `npm install` — установка зависимостей
- `npm run build` — сборка проекта
- `npm run dev` — разработка проекта
- `npm run lint` — линтер
- `npm run start` — запуск локальной версии на http://localhost:3000/
